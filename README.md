# Smartly Hire (AI Proctor Interview System)

## Requirements
- Windows 10/11
- Node.js LTS (18 or 20 recommended)
- MongoDB Atlas connection string (or local MongoDB)
- (Optional) Ollama + llama3 for question generation/evaluation

---

## 1) Setup Environment Files

### Server env
1. Go to `server/`
2. Copy `.env.example` and rename it to `.env`
3. Fill values inside.

### Client env
1. Go to `client/`
2. Copy `.env.example` and rename it to `.env`
3. Fill values inside.
